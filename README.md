# Data Structures in C

This repository contains implementations of fundamental **Data Structures and Algorithms in C**.  
It is intended for learning, practice, and revision of core concepts used in computer science.

The code is written with clarity in mind and follows a step-by-step, beginner-friendly approach.



## 📌 Contents

### 🔍 Searching Algorithms
- Linear Search
- Binary Search

### 🔃 Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort

### 🔗 Linked Lists
- Singly Linked List Operations
- Singly Linked List Concatenation

(More data structures and algorithms will be added gradually.)
